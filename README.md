# Pharmacy Backend

Creating a Pharmacy Backend for learning involves user authentication, patient and medication record management, prescription tracking, and billing. Implement secure features for admins and users, tracking medication dispensing and pricing. Ensure an intuitive user interface and prioritize data security. Define clear learning objectives for a focused development approach. Regularly reflect on challenges encountered for optimal learning.

# Clone the repository:

```
git clone https://github.com/your-username/server.git
cd server
```

# Install Dependencies:

```

npm install
Set Environment Variables:
Create a .env file in the root of your project and add the necessary environment variables. Refer to .env.example for a template.
```

# Start the Server:

```

npm start

Once the installation steps are complete, you can access the server at http://localhost:your-chosen-port.
```

# Dependencies to download

```

bcryptjs: Library for hashing passwords.
cloudinary: Cloud storage for images.
cookie-parser: Parse cookies.
cors: Enable CORS (Cross-Origin Resource Sharing).
dotenv: Load environment variables.
express: Web application framework.
jsonwebtoken: JSON Web Token (JWT) implementation.
mongoose: MongoDB object modeling.
morgan: HTTP request logger middleware.
multer: Middleware for handling multipart/form-data.
nodemailer: Send emails.
passport: Authentication middleware.
```

Make sure to review each package's documentation for more details on how to use them.
